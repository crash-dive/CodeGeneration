What Is Code Generation?

  • Not a compiler because there is no standard language spec
  • Not reflection or metaprogaming because it is not done at runtime
  • Code that writes code to a file before runtime

What Can We Use To Do It?

  • T4
    • Microsoft 'supported'
    • Minimal visual studio support
    • Hard to integrate into CI
  • Scripty
    • Open source community supported
    • Ok visual studio support
    • Can to integrate into CI via Cake or MSBuild
  • Code Smith Generator
    • Commercially supported
    • Good visual studio support
